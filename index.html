如何替换为你自己的图片？

1️⃣ 把你的图片上传到这个仓库：点击页面上方 Add file → Upload files，把图片拖进去，然后 Commit 保存。

2️⃣ 上传后点进图片文件，点击右上角 Download / View raw，复制图片的链接。

3️⃣ 回到 index.html 编辑，把上面示例里的 src="https://via.placeholder.com/600x400?text=Image+1" 换成你自己的图片链接就可以了。

4️⃣ 想多放几张，只要复制一整块 <div class="card">...</div> 再改标题和描述。
